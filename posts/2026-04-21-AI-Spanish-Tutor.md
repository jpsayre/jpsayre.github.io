# Using AI as a Spanish Tutor

I always love finding new use cases for how I can use AI. Recently I've been using a Claude Cowork project to teach me Spanish. I'll breakdown my approach:

1) Setup a new project in Claude Cowork. It's important that this isn't a normal chat, as it will need access to read and write files.
2) Start a new chat and have Claude test you on your Spanish (or any language). I did this by translating sentences and paragraphs it provided in English to Spanish. This was enough for it to understand my level.
3) I asked for it to build me a roadmap for learning Spanish based on where I was currently at. It quickly and accurately diagnosed my deficiencies and plotted out the order in which I should address them.
4) I had Claude write its own prompt. It took this roadmap, my current level, and my preferences and saved them in a markdown file for it to reference.
5) I setup a status file for Claude to keep track of my progress. I also created a CSV file for Claude to store vocabulary that I didn't know along with its English translation. I can import this directly into the flashcard app of my choice (ANKI) to study the words and phrases that I haven't known when translating exercises.
6) I now complete the drills that Claude gives me and it automatically tracks my progress, updates my vocab, uses the vocab I miss more frequently, and determines when I'm ready to move to the next level in our roadmap.

I've been using this system for a few weeks and I've already noticed a real difference in both my grammar and vocab. Particularly an improvement with using natural sounding phrases versus more literal translations.

I actually still have a spanish tutor and believe that AI is supplementing that, but now we get to focus more on speaking practice rather than getting bogged down in grammar. I think it's a win for both of us.
